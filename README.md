# MWECAU Election Platform

A secure and easy-to-use digital voting system built for Mwenge Catholic University (MWECAU) student elections.

## Project Structure

```
mwecau_election_platform/
├── docs/                    # Documentation files
│   ├── database_design.md
│   └── MWECAU-Voting-Guide.md
├── src/                     # Source code
│   ├── frontend/            # HTML, CSS, and client-side code
│   ├── backend/             # Server logic, API, and database
│   └── full_stack/          # Combined app (ready for deployment)
├── README.md                # This file
├── LICENCE.txt              # Proprietary license
└── path.txt                 # Project path info
```

## Technology Stack

- Frontend: HTML5, CSS3, JavaScript
- Backend: Python (Django), REST API
- Database: MySQL / PostgreSQL / SQLite
- Tools: Git, Python Virtual Environment
- Deployment : Docker, Gunicorn, Nginx  

## Quick Start

### 1. Clone the repository
```bash
git clone https://github.com/mwecau-ict-club/mwecau_election_platform.git
cd mwecau_election_platform
```

### 2. Choose your development approach:
- Full Application: Navigate to [`src/full_stack/`](src/full_stack/README.md) and follow its README
- Frontend Only: Check [`src/frontend/README.md`](src/frontend/README.md)
- Backend Only: Check [`src/backend/README.md`](`src/backend/README.md`)
